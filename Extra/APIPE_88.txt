Insatll swagger UI as a library:

All modules location is : /opt/apigee/data/apigee-drupal-devportal/sites/all/modules/

Pretasks: Chnage the directory to /opt/apigee/apigee-drupal/wwwroot

step 1 : Download the appropriate Swagger UI version at https://github.com/swagger-api/swagger-ui/releases
step 2: Extract the archive and rename the folder from "swagger-ui-master" to "swagger_ui"
step 3: Place the renamed folder in the [DRUPAL ROOT]/libraries directory so its path will be [DRUPAL ROOT]/libraries/swagger_ui   /opt/apigee/data/apigee-drupal-devportal/sites/all/libraries
step 4: Install the module from https://ftp.drupal.org/files/projects/swagger_ui_formatter-7.x-2.0.tar.gz or use
